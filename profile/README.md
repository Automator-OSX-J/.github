## Overview

<a href="#">Automator</a> is Apple's built-in automation tool that lets Mac users build powerful workflows without any programming knowledge. By connecting pre-built actions in a visual editor, repetitive tasks that would take minutes or hours manually complete automatically in seconds. The application ships with macOS and integrates deeply with system functions and Apple's own applications.

The <a href="#">action library</a> contains hundreds of pre-built steps covering file operations, image processing, text manipulation, application control, and system functions. Browse actions by category or search for specific operations. Drag actions into the workflow canvas and connect them in sequence. <a href="#">Data flows</a> between actions automatically — the output of one step becomes the input of the next, creating processing pipelines without any configuration of connections.

<a href="#">Workflow types</a> determine how automations run. Standard workflows run when opened in Automator. Applications save as standalone double-clickable programs. Folder Actions attach to specific folders and trigger automatically when files are added. <a href="#">Services integrate</a> into the right-click menu of other applications, enabling context-sensitive automation on selected text, files, or other content. Print Plugins add custom steps to the print dialog. Calendar Alarms run workflows at scheduled times.

The <a href="#">variable system</a> stores and retrieves values throughout workflows, enabling dynamic behavior beyond simple linear processing. Ask for user input with dialog boxes, store results in variables, and reference those values in later actions. <a href="#">Conditionals and loops</a> via shell scripts or AppleScript actions embedded in workflows handle branching logic for more sophisticated automations. These capabilities bridge the gap between simple action sequences and genuinely programmable workflows.

## Key Features

- <a href="#">Hundreds of built-in actions</a> cover file operations image processing text manipulation and app control
- Visual <a href="#">drag-and-drop workflow builder</a> connects actions without writing a single line of code
- <a href="#">Multiple workflow types</a> run as apps folder actions services calendar alarms or print plugins
- <a href="#">Folder Actions</a> trigger workflows automatically whenever new files are added to watched folders
- <a href="#">Variable storage system</a> saves and reuses values throughout workflows for dynamic processing logic
- <a href="#">AppleScript integration</a> embeds custom scripts into workflows for operations no built-in action covers
- <a href="#">Shell script support</a> brings full terminal command power directly into visual workflow pipelines
- <a href="#">Batch image processing</a> resizes crops converts and watermarks large photo collections automatically
- <a href="#">Service workflows</a> add custom right-click menu actions to any Mac application for selected content
